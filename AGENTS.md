Theseus translates 32-bit Windows x86 code to Rust.  This allows us to recompile
it to a new target architecture.

## Project layout

tc/: exe to Rust generator
runtime/: runtime support for generated binary
win32/pe/: PE reading
win32/winapi/: implementation of Windows kernel32 functions
exe/: generated exe code (output of tc) for a few test executables

Ignore the content under exe/ because it is autogenerated.  Feel free to make changes
that break the API expected by those files.

## x86 emulation limitations

This project doesn't support the AF flag yet, don't report it as a bug.

In general, when looking for bugs, hitting todo!()s is not a bug, because it
will cause an obvious crash with a call stack.  If we're attempting to track down
a bug about a todo I will ask for it directly.
